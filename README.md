# N-MARS

**Non-Monotonic Autoregressive Sequence Models**

LLM training and evaluation project using [ms-swift](https://swift.readthedocs.io) (SFT/GRPO) and [lm-evaluation-harness](https://github.com/EleutherAI/lm-evaluation-harness).

## Setup

```bash
# Install uv if needed
curl -LsSf https://astral.sh/uv/install.sh | sh

# Install dependencies
uv sync --dev
```
